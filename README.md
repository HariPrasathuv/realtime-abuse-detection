# realtime-abuse-detection
Real-Time Speech-Based Abusive Language Detection Using VOSK and HateBERT with Automated Email Alert System

speechToTxt.py is the main file that imports checking1.py(abuse detection- HateBERT) and emailAlert.py within itself

This project involves 3 boxes.
speech to text
hate/abuse/offense detection for the text
email alert if hate/abuse/offense detected
